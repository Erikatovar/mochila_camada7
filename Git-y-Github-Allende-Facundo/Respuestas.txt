a. ¿Quién inventó el sistema de control de versión Git y por qué?
Diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente
Se creo basicamente para adaptar la forma en que se venian guardando versiones de codigo, con el fin de conseguir una forma mas ordenada y practica de guardar los archivos de codigo, Linus Torvalds lo creo para controlar el desarrollo del kernel de Linux
