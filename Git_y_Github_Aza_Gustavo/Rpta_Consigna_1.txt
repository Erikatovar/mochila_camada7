¿Quién inventó el sistema de control de versión Git y por qué?
RTA: Desarrollado por Linus Torvalds, debido a la Complejidad de los sistemas de versiones de la época (2005)
¿A quién pertenece actualmente Github y por qué?
RTA: Microsoft compró Github el 04/jun/2018. Por una adopción de la compañía a una estrategia nueva, mucho más abierta al mundo del softwar libre y a la integración con otras plataformas y servicios

RTA: Online de manera gráfica en la página: github.com, usando las diferentes funciones como clonar, nuevo repositorio, agregar archivos, entre otras.
