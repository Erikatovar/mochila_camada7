a) ¿Quien inventó el sistema de control de version Git y por qué?
    Fue diseñado por Linus Torvalds. Pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones, cuando estas tienen un gran número de archivos de código fuente. Su propósito llevar el registro de los cambios en archivos de computación; incluyendo coordinar el trabajo de varias personas.
